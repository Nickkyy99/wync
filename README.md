# Wync

*simple messages from people who matter.*

This public repository hosts the Wync marketing website:

[https://nickkyy99.github.io/wync/](https://nickkyy99.github.io/wync/)

Wync is a macOS menu bar app for small private nudges from people who matter.
Someone you love opens a private sender link, types a short note, and the words
appear quietly in your Mac's menu bar.
